# FriendFinder
A fullstack app using node and express. This is a survey type website that determines what character you are most like from the show "It's always Sunny in Philadelphia. This website is completely responsive on all screen sizes, and there are some cool UI components. Styled using Bootstrap 4 & CSS. Delpoyed through heroku:
